 Автоматизированные тесты Playwright — 3S.INFO Events Widget

Репозиторий содержит E2E-тесты для страницы конструктора календаря мероприятий:

https://dev.3snet.info/eventswidget/

Тесты реализованы на JavaScript с использованием **Playwright Test**  
и выполняются в браузере **Firefox**.



 Что проверяется

 Базовая загрузка страницы

- страница открывается без ошибок
- присутствует заголовок  
  «Начните создавать свой календарь мероприятий!»
- отображаются шаги 1–4 мастера настройки
- видны элементы управления и кнопки

 Happy-path сценарий

- выбор тематик
- выбор стран
- ввод размеров блока (ширина / высота)
- выбор цветовой темы
- генерация превью
- наличие списка событий
- кнопка «Скопировать код» доступна

 Негативный сценарий

- тематика не выбрана
- генерация не ломает страницу
- интерфейс остаётся работоспособным


 Запуск локально

Необходим Node.js 18+

```bash
npm install
npm test
